üßæ After Admission: Foundation Level

Once admitted, students must complete the Foundation Level as the first phase.

It includes 8 subjects (each 4 credits):

  • Mathematics 1 & 2
  • Statistics 1 & 2
  • Computational Thinking
  • English 1 & 2
  • Programming in Python

After completion, students can:

  • Continue to the Diploma/Degree phase
  • Exit with a Foundation Certificate
üíª Diploma in Programming

After completing the Foundation Level, students can choose to pursue the Diploma in Programming.

The following courses are part of the diploma:

  • Database Management Systems
  • Programming, Data Structures and Algorithms using Python
  • Modern Application Development I
  • PROJECT: Modern Application Development I - Project
  • Programming Concepts using Java
  • Modern Application Development II
  • PROJECT: Modern Application Development II - Project
  • System Commands

After completing all courses, students earn the Diploma in Programming from IIT Madras.

üìä Diploma in Data Science

Students can also choose to pursue the Diploma in Data Science after completing the Foundation Level.

The following courses are part of this diploma:

  • Machine Learning Foundations
  • Business Data Management
  • PROJECT: Business Data Management - Project
  • Machine Learning Techniques
  • Machine Learning Practice
  • PROJECT: Machine Learning Practice - Project
  • Business Analytics
  • Tools in Data Science

Upon successful completion, students earn the Diploma in Data Science from IIT Madras.

üéì BSc Degree in Programming & Data Science

After completing both diplomas, students can advance to the BSc Degree Level.

Courses in this phase include:

  • Software Engineering
  • Software Testing
  • AI: Search Methods for Problem Solving
  • Deep Learning
  • Strategies for Professional Growth

Upon completion, students receive the BSc Degree in Programming & Data Science from IIT Madras.

üéØ Final Phase: BS Degree in Data Science & Applications

Students may choose to pursue the BS Degree for deeper expertise and applied knowledge.

Courses in this final level build upon foundational, programming, and data science skills to apply them in real-world scenarios.

Graduates of this track receive the BS Degree in Data Science and Applications from IIT Madras.
